# Time Series Forecasting App

## Overview

This project predicts future airline passenger counts using the ARIMA Time Series Forecasting model.

## Features

- Forecast future passenger numbers
- Interactive Streamlit interface
- Adjustable forecasting horizon
- Forecast visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Statsmodels
- Streamlit
- Joblib

## Model

ARIMA (AutoRegressive Integrated Moving Average)

## Run Locally

pip install -r requirements.txt

streamlit run app.py
